Что внутри

  • ✅ SubAgent Manager (7 ролей, mpsc)
  • ✅ Инструменты: read_file, write_file, grep_files, exec_shell
  • ✅ LLM клиент (Ollama / DeepSeek)
  • ✅ MCP Client (подключение MCP-серверов)
  • ✅ Сессии (save/resume контекста)
  • ✅ Автономия L0-L4 (работа без связи)
  • ✅ DTN Engine (tc-netem, эмуляция задержек)
  • ✅ Node Protocol (UUID, discovery, announce)
  • 🟡 Kafka Transport (опционально, для роя)
