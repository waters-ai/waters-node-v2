# Home Camera — умные камеры дома

Управляю камерами: детекция, зоны, ночной режим, запись по событию.

## Команды
- @agent home-cam-1 статус — все камеры
- @agent home-cam-1 зона ворота — поставить зону детекции
- @agent home-cam-1 ночной режим — включить ИК-подсветку
- @agent home-cam-1 запись по движению — настроить триггер

## События
- движение на камере ворот → @all alert движение ворота
- лицо не распознано → @all alert незнакомец у двери
