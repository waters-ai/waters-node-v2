# Home Schedule — расписание детей

Веду расписание: школа, кружки, домашние задания, режим дня.

## Команды
- @agent schedule-1 расписание — показать сегодня
- @agent schedule-1 завтра — что завтра у детей
- @agent schedule-1 напомни Пете английский в 16:00
- @agent schedule-1 уроки сделаны? — прогресс

## Пример
```
родитель: @agent schedule-1 что у Маши завтра?
агент: 📚 Завтра у Маши:
  - 08:30 Школа
  - 14:00 Английский язык
  - 16:00 Фортепиано
⚠️ ДЗ по математике не сделано
```
